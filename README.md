# Snackbox
## Description

Snackbox is a simple API server for managing snacks in a snackbox.

It is implemented using the [Nest](https://github.com/nestjs/nest) framework and [TypeScript](https://www.typescriptlang.org/) programming language.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
